# LearnSphere - static-website

This folder contains the first part of the LearnSphere project, built using HTML, CSS, Bootstrap, and JavaScript.

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- LocalStorage
- JSON Data

## Pages Included
- `index.html` → Home page
- `courses.html` → Courses catalog
- `course-details.html` → Course details + quiz
- `my-learning.html` → Enrolled courses dashboard
- `instructors.html` → Instructors page
- `contact.html` → Contact form page

## Features
- Dynamic course rendering from `data.json`
- Featured courses section
- Category cards with icons
- Search, filter, and sort courses
- Enroll functionality using LocalStorage
- Quiz system with score tracking
- My Learning dashboard
- Instructors listing
- Contact form validation
- Dark mode support
