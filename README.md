# LearnSphere Project

This repository contains two parts of the LearnSphere project:

## 1) JavaScript Version
Folder: `static-website/`

Built using:
- HTML
- CSS
- Bootstrap
- JavaScript

Pages included:
- Home
- Courses
- Course Details
- My Learning
- Instructors
- Contact

Features:
- Course filtering
- Enrollment using localStorage
- Quiz system
- Dark mode
- Contact form validation

## 2) Angular Version
Folder: `my-learnsphere/`

Built using:
- Angular 16
- TypeScript

Main features:
- Courses list
- Course card component
- Enrollment dashboard
- Routing
- Service-based data handling
- Live filtering
- Category filtering
- Event binding / property binding / interpolation / ngIf / ngFor / ngSwitch

## Notes
- The live deployment is for the JavaScript version only.
- The Angular version is intended for local demo during evaluation.
