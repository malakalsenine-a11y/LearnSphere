# LearnSphere - Angular Version

This folder contains the Angular SPA version of the LearnSphere project.

## Technologies Used
- Angular 16
- TypeScript
- HTML
- CSS

## Components
- NavbarComponent
- CoursesListComponent
- CourseCardComponent
- EnrollmentDashboardComponent

## Main Features
- Display all courses using *ngFor
- Search courses live using (input) event binding
- Filter courses by category
- Highlight selected category using [class.active]
- Pass course data using @Input()
- Emit enroll event using @Output()
- Handle course enrollment in dashboard
- Show enrolled course count in navbar using interpolation
- Show "Coming Soon" banner using *ngIf
- Use ngSwitch for course level badges
- Routing between /courses and /dashboard

## Notes
- This Angular project is intended for local demo during evaluation.
- It is not deployed as a live link.
- The project follows the assignment requirements for Angular SPA.

## Run the Project
Use:
ng serve -o
